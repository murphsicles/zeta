# Zeta Compiler v0.3.53 - Dependency Optimization & Clean Build
**Release Date:** 2026-04-03

## What's Awesome About This Release

This release focuses on build system optimization and dependency management, temporarily disabling the `nour` Bitcoin SV dependency to streamline testing and reduce build complexity. We've achieved a cleaner build environment while maintaining compiler functionality, paving the way for more efficient development workflows.

## Migration Notes

This release maintains full backward compatibility with v0.3.52 for core compiler functionality. The `nour` Bitcoin SV dependency is temporarily disabled but can be re-enabled when blockchain features are required.

**Note for Blockchain Developers**: Blockchain features requiring `nour` are temporarily unavailable. Use v0.3.52 for blockchain development or await v0.3.55+ for restored blockchain support.
