#### **1. Fundamental Semirings (Built-in):**
1. **Natural numbers semiring** (ℕ, +, ×, 0, 1)
   - Additive monoid: (ℕ, +, 0) commutative
   - Multiplicative monoid: (ℕ, ×, 1) commutative
   - Distributivity: a × (b + c) = (a × b) + (a × c)

2. **Integer semiring** (ℤ, +, ×, 0, 1)
   - Extends naturals with additive inverses
   - Forms a ring (additive group)

3. **Boolean semiring** ({0,1}, ∨, ∧, 0, 1)
   - Logical OR as addition (∨, identity 0)
   - Logical AND as multiplication (∧, identity 1)
   - Distributive: a ∧ (b ∨ c) = (a ∧ b) ∨ (a ∧ c)

4. **Tropical semiring** (ℝ ∪ {∞}, min, +, ∞, 0)
   - "Min-plus" algebra for shortest paths
   - Addition: min(a, b)
   - Multiplication: a + b
   - Identity: ∞ for min, 0 for +

5. **Arctic semiring** (ℝ ∪ {-∞}, max, +, -∞, 0)
   - "Max-plus" algebra for longest paths
   - Dual of tropical

6. **String semiring** (Σ*, concatenation, union, ε, ∅)
   - For formal language theory
   - Addition: union of languages
   - Multiplication: concatenation
   - Identity: empty string ε

7. **Set semiring** (𝒫(S), ∪, ∩, ∅, S)
   - Power set with union and intersection
   - Fundamental for relational algebra

8. **Matrix semiring** (M_n(R), +, ×, 0, I)
   - Matrices over another semiring R
   - Enables linear algebra over any semiring
