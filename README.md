## **🧪 TECHNICAL DETAILS**

### **Language Features Implemented:**
- Functions with parameters and return types
- Variables and assignments
- Arithmetic and logical operators
- Control flow (if/else)
- Basic type system (i64)
- Recursion (no loops in v0.3.7 constraints)

### **Compiler Architecture:**
- **Lexer**: Tokenizes source code
- **Parser**: Builds abstract syntax tree  
- **Generator**: Produces executable code
- **Optimizer**: Basic optimizations
- **Linker**: Creates final binaries

### **Limitations (v0.5.0):**
- Simplified syntax (working within v0.3.7 constraints)
- No strings or arrays (integer codes only)
- No file I/O (simulated)
- Basic error handling
